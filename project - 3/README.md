✊✋✌️ Rock-Paper-Scissors Game (Python)
📌 Project Overview

This is a simple Rock-Paper-Scissors game built using Python. The user plays against the computer, and the winner is decided based on standard game rules.

🎮 Features
👤 User Input: Choose between Rock, Paper, or Scissors
🤖 Computer Choice: Random selection by the system
⚖️ Game Logic:
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
🏆 Result Display: Shows both choices and declares win/lose/tie
🔁 Play Again Option: Allows multiple rounds
📊 Score Tracking (Optional): Keeps track of user and computer scores
🖥️ Simple User Interface: Easy-to-use command-line interaction

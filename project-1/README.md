🧮 Simple Calculator in Python
📌 Project Overview

This project is a basic calculator developed using Python. It performs simple arithmetic operations based on user input, making it a great beginner-friendly program to understand conditional statements and user interaction.

🚀 Features
Accepts two numerical inputs from the user
Supports basic arithmetic operations:
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
Handles invalid input gracefully
Prevents division by zero errors
🛠️ Technologies Used
Python (Core Concepts)
Command Line Interface (CLI)
📂 How It Works
User enters two numbers
Selects an operation (1/2/3/4)
Program performs the calculation
Result is displayed instantly
🎯 Learning Outcome
Understanding of conditional statements (if-elif-else)
Handling user input in Python
Basic error handling techniques
